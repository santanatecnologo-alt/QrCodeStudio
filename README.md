# QR Code Studio 🚀

Um gerador de QR Code moderno, rápido e inteligente, desenvolvido puramente com **HTML5, CSS3 e JavaScript**. O projeto permite a criação de códigos personalizados para diversos fins, com formatação automática de dados.

## 🔗 Demonstração
Acesse o projeto online: [https://santanatecnologo-alt.github.io/QrCodeStudio/](https://santanatecnologo-alt.github.io/QrCodeStudio/)

## ✨ Funcionalidades
- **Formatação Inteligente**: 
  - 💬 **WhatsApp**: Gera links diretos `wa.me` apenas com o número.
  - 💸 **PIX**: Compatível com códigos "Copia e Cola".
  - 📞 **Telefone**: Abre o discador automaticamente.
  - 🌐 **URLs**: Garante o protocolo `https://`.
    
- **Personalização**:
  - Ajuste de cores em tempo real.
  - Seleção de dimensões (128px, 256px, 512px).
- **Download**: Opção de baixar o código gerado em formato `.png`.

## 🛠️ Tecnologias Utilizadas
- **HTML5/CSS3**: Estrutura e design responsivo.
- **JavaScript (Vanilla)**: Lógica de manipulação de dados e DOM.
- **QRCode.js**: Biblioteca leve para renderização dos códigos.

## 📱 Observação Mobile
Para melhor experiência de download em dispositivos móveis, caso o navegador bloqueie o download automático, recomenda-se:
1. Pressionar e segurar sobre a imagem gerada.
2. Selecionar "Fazer download da imagem" ou "Salvar em Fotos".
3. Alternativamente, utilize o modo "Para Computador" no navegador do celular.
